## Install

Available from the [Mozilla Firefox Addon Store](https://addons.mozilla.org/en-US/firefox/addon/hacker-news-reading-time/) or Chrome Webstore (TODO).

## Development

1. clone repo
2. npm install
3. npm run start:firefox
